# Student Risk Prediction

This project predicts student dropout risk using machine learning techniques.

## 📊 Dataset
The dataset contains student information such as grades, attendance, and demographics.

## ⚙️ Model
A classification model was trained to identify students at risk.

## 🚀 Results
The model achieved good performance in predicting at-risk students.

## 📌 Tools
- Python
- Pandas
- Scikit-learn
